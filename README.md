## 🏃‍♂️ Running Locally

To run this project on your local machine:

```bash
# Clone the repository
git clone https://github.com/barcebalhenna/my-portfolio-va.git

# Navigate into the directory
cd my-portfolio-va

# Install dependencies
npm install

# Start the development server
npm run dev
```
